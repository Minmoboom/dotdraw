# DotDraw — AI 创意生成平台

## 📌 项目概述

**DotDraw** 是一个在线网站（Web 应用），用户通过浏览器访问网址，注册登录后使用。用户用自然语言输入需求，平台调用 AI 模型自动生成**图片、视频、PPT 幻灯片**等内容。

**核心特点：**
- 🌐 部署在服务器上，有公开网址，任何人在浏览器中都能访问
- 👤 用户需要注册账号、登录后才能使用
- 💬 左侧对话式交互，用户输入需求、AI 返回结果
- 🖼️ 右侧案例展示区，由管理员配置

**参考网站：** [dotdraw.art](https://dotdraw.art)

---

## 🎯 产品目标

让没有任何设计技能的用户，通过简单的文字描述，就能创建专业级的视觉内容。

## 🏗️ 技术架构

| 组成部分 | 技术选择 | 说明 |
|----------|----------|------|
| 前端框架 | Next.js (React) | 页面和交互 |
| 后端服务 | Supabase | 用户登录 + 数据库 |
| AI 服务 | 阿里云通义万相 | 图片生成（国内可用） |
| 部署平台 | Vercel | 免费托管 |
